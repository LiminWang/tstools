NAME

bincat - manual page for bincat 1.0.0  

SYNOPSIS

bincat [OPTION] file [OPTION]  

DESCRIPTION

'bincat' read binary file, translate 0xXX to 'XY ' format, then send to stdout.  

OPTIONS

-a, --addr
show data address at line head, default: do NOT show it
-d, --decaddr
dec address format, default: hex
-s, --seperate <,>
white space, any char except [0-9A-Fa-f], default: ' '
-w, --width <w>
w-byte per line, [1,10922], default: 188
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

bincat xxx.ts
