NAME

tsana - manual page for tsana 1.0.0  

SYNOPSIS

tsana [OPTION]...  

DESCRIPTION

'tsana' get TS package from stdin, analyse, then send the result to stdout.  

OPTIONS

-pid-list
show PID list information, default option
-psi-tree
show PSI tree information
-outpsi
output PSI package
-pid <pid>
set cared <pid>, default: ANY PID
-prog <prog>
set cared <prog>, default: ANY program
-interval <iv>
set cared <iv>(ms) for bit-rate calculate, default: 1000
-pcr
show PCR information of cared <pid>
-rate
output bit rate of PID of cared <program>
-ptsdts
output PTS and DTS information of cared <pid>
-pes
output PES data of cared <pid>
-es
output ES data of cared <pid>
-err
show all errors found
-mono
disable colour effect, default: use colour to help read
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

"bincat xxx.ts | tsana -err" -- report all error information
