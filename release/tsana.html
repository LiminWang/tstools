NAME

tsana - manual page for tsana 1.0.0  

SYNOPSIS

tsana [OPTION]...  

DESCRIPTION

'tsana' get TS package from stdin, analyse, then send the result to stdout.  

OPTIONS

-pid-list
show PID list information, default option
-psi-tree
show PSI tree information
-outpsi
output PSI package
-time
show time instead of address when TS error
-cc
check Continuity Counter
-pcr
show all PCR value
-pid <pid>
set cared <pid>
-pes
output PES data of <pid>
-es
output ES data of <pid>
-ptsdts
output PTS and DTS of <pid>
-h, --help
display this information
-v, --version
display my version
 

EXAMPLES

"bincat xxx.ts | tsana -cc" -- report CC error information
